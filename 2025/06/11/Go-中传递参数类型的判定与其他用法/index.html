Go_中传递参数类型的判定与其他用法

总结

  1. 获取类型对象：reflect.TypeOf(v) → 获取类型信息
  2. 检查指针类型：t.Kind() == reflect.Ptr → 是否为指针
  3. 解引用指针：t = t.Elem() → 获取指针指向的实际类型
  4. 获取类型名称：t.Name() → 类型的字符串名称

关键点：

  • Elem() 用于解引用指针，获取其指向的类型
  • Name() 返回类型的名称，对于命名类型有效
  • 对于指针类型，需要解引用后才能获取实际类型的名称
  • 匿名类型和基本类型处理需要特别注意
