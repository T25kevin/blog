发表于 2023-05-22 更新于 2024-06-08 分类于 免费项目

云

IaaS指提供系统（可以自己选）或者储存空间之类的硬件，软件要自己手动装；PaaS提供语言环境和框架（可以自己选）；SaaS只能使用开发好的软件（卖软件本身）；BaaS一般类似于非关系数据库，但各家不通用，有时还有一些其它东西。

2. Paas

  • https://www.heroku.com/ java go py docker。国内访问不佳
  • https://fly.io/docs/pricing/ py node go 静态，感觉很完美
  • https://www.deta.sh/ py3.7 node12 内存128MB，依赖250MB，大概率不支持DBMS，支持定时任务。有类dict数据库和10GB存储且可独立使用
  • https://www.pythonanywhere.com/ 限制非常多，几乎就和虚拟主机差不多了，免费账户不允许访问白名单之外的网站。但好歹能提供一个自动https的web app

  • 机器学习
  • Docker Registry：储存0.5GB，流量5GB

debian下配置ftp服务器：

由于开发需要使用ftp功能，一时之间有点蒙，这玩意一直都会用，但怎么配置还是有点蒙，然后花了两个小时去恶补了下相关知识！

安装ftp：

直接使用命令安装即可

sudo apt install vsftpd
# 重点：这里是vsftpd不是ftp！！！

ftp服务启动：

第一种方式：

/etc/init.d/vsftpd {start|stop|restart|reload|status}
# 重点：中括号前面的不变，后面的是命令

第二种方式：

service vsftpd {start|stop|restart|reload|status}
# 重点：中括号前面的不变，后面的是命令

第三种方式：

systemctl {start|stop|restart|reload|status} vsftpd.service
# 重点：只有中括号中的命令在变
# 个人更喜欢第三种，简单粗暴，有深度！

开机自启：

自动启动：

systemctl enable vsftpd.service

取消自动启动：

systemctl disable vsftpd.service

修改配置：

apt方式安装的ftp，配置文件默认在’/etc/vsftpd.conf

vim /etc/vsftpd.conf
