debian下配置ftp服务器：

由于开发需要使用ftp功能，一时之间有点蒙，这玩意一直都会用，但怎么配置还是有点蒙，然后花了两个小时去恶补了下相关知识！

安装ftp：

直接使用命令安装即可

sudo apt install vsftpd
# 重点：这里是vsftpd不是ftp！！！

ftp服务启动：

第一种方式：

/etc/init.d/vsftpd {start|stop|restart|reload|status}
# 重点：中括号前面的不变，后面的是命令

第二种方式：

service vsftpd {start|stop|restart|reload|status}
# 重点：中括号前面的不变，后面的是命令

第三种方式：

systemctl {start|stop|restart|reload|status} vsftpd.service
# 重点：只有中括号中的命令在变
# 个人更喜欢第三种，简单粗暴，有深度！

开机自启：

自动启动：

systemctl enable vsftpd.service

取消自动启动：

systemctl disable vsftpd.service

修改配置：

apt方式安装的ftp，配置文件默认在’/etc/vsftpd.conf

vim /etc/vsftpd.conf

我的配置如下：

#

xferlog_enable=YES

# Make sure PORT transfer connections originate from port 20 (ftp-data).
connect_from_port_20=YES

#chroot_local_user=YES

命令：ps -ef|grep详解

  • ps命令将某个进程显示出来
  • grep命令是查找
  • 中间的|是管道命令 是指ps命令与grep同时执行
  • PS是LINUX下最常用的也是非常强大的进程查看命令


grep命令是查找，是一种强大的文本搜索工具，它能使用正则表达式搜索文本，并把匹配的行打印出来。

grep全称是Global Regular Expression Print，表示全局正则表达式版本，它的使用权限是所有用户。
