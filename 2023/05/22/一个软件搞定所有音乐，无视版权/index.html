一个软件搞定所有音乐，无视版权

发表于 2023-05-22 更新于 2025-06-09 分类于 _A8-工具 ， 音乐播放器

放松心情的时候，往往都想来上一首歌，但是市面上的音乐软件，版权杂乱，往往一个音乐软件是找不到自己所想要听的歌。

现在老夫来带一款来自github的开源软件 – Listen 1

来拯救你那无处安放，有骚动的心！软件最大特点：解锁鹅厂、某易云、某我. . .等市面所有音乐软件因版权，而发黑的链接！达到可以直接播放的目的，也就是，以前你听歌需要N歌软件，那么现在，你只需要一个就够了！

遗憾的是：软件目前功能稍有BUG，当然，这些bug是对于懂程序的人来所，如果你就单纯的听音乐。你可能无法发现这些bug。所有你可以放心食用

  • 浏览器插件：
    • Firefox插件：点击进入Firefox插件中心
    • Microsoft Edge插件：点击进入Microsoft Edge插件中心
    • Chrome 插件版 V2.21.6：点击下载
  • 客户端下载：
    • 安卓：点击下载
    • Windows 桌面版安装包：点击下载
    • Mac Universal 桌面版：点击进入下载页面
    • Linux 64位桌面版：点击下载
    • Windows 桌面绿色版64位：点击下载
    • Windows 桌面绿色版32位：点击下载 界面是这样的：

安装：

  • Chrome插件版
    • 下载Chrome插件版压缩包，并解压缩
    • 打开 Chrome 扩展页面，勾选右上角「开发者模式」
    • 选择「加载已解压的扩展程序…」，选择刚刚解压缩的文件夹
    • 开始使用
  • Firefox插件版
    • 下载Firefox插件版，并解压缩
    • 打开 about:config 页面，xpinstall.signatures.required 设置为 false
    • 选择「附加组件」，齿轮图标，选择从文件安装附加组件，选择下载的xpi文件
    • 开始使用

  • 至于windos和linux 就不说了！linux安装软件不会，你还是用回windos吧！

  • 至于. . . windos不会的. . .老夫就爱莫能助了！
