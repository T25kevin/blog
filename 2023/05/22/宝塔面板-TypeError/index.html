宝塔面板_TypeError

发表于 2023-05-22 更新于 2024-06-26 分类于 linux

宝塔面板_TypeError：

关于宝塔面板下的 TypeError: ‘NoneType’ object is not subscriptable 错误

说明：该错误的引发原因是因为安装插件 导致/www/server/panel/data/plugin.json 丢失引起的

解决方法一：

  • 重装宝塔面板

解决办法二：

  • 去另一个宝塔面吧下面，复制一份 plugin.json 丢到制定文件夹下即可

相对于已经稳定运行了很久的服务器来说， 重装肯定是不可能的，所以无疑第二种是最优的解决方案。
