AIжҚўи„ёйЎ№зӣ® - faceswap еҹәдәҺCentos7 зҡ„йғЁзҪІгҖҒй…ҚзҪ®гҖҒи®ӯз»ғгҖҒжӣҝжҚўж•ҷзЁӢпјҲеҺҹеҲӣпјү



зҹҘиҜҶжӢ“еұ•пјҡ

  • гҖҗзІҫйҖүгҖ‘Nvidia GPU жңҖж–°и®Ўз®—иғҪеҠӣиЎЁпјҲCUDA Compute Capabilityпјү_cudaи®Ўз®—иғҪеҠӣжҺ’иЎҢ_Dust_Evcзҡ„еҚҡе®ў-CSDNеҚҡе®ў

  • и®ӯз»ғжҢҮеҚ—-deepfakes/faceswapпјҡи®ӯз»ғFaceswapпјҢжЁЎеһӢжҒўеӨҚе’ҢеҲҶжһҗпјҢжүӢжҠҠжүӢж•ҷеӯҰпјҢз®ҖеҚ•еҝ«йҖҹдёҠжүӢпјҒ - зҹҘд№Һ (zhihu.com)

  • python - faceswapжҚўи„ёзЁӢеәҸе®үиЈ…еҸҠдҪҝз”Ё - дёӘдәәж–Үз«  - SegmentFault жҖқеҗҰ

зҺҜеўғиҜҙжҳҺпјҡ

  • зі»з»ҹпјҡCentos7.9
  • Intel(R) Xeon(R) CPU E5-2678 v3
  • 2йғЁзҪІж–№ејҸпјҡCPU
  • жҢҮд»ӨйӣҶејәеҲ¶иҰҒжұӮпјҲCPUж–№ејҸе®үиЈ…пјүпјҡAVX
  • иҪҜд»¶пјҡFFmpeg
  • pythonзүҲжң¬пјҡ3.10пјҲејәеҲ¶иҰҒжұӮпјҢйқһ3.10пјҢеҫҲеӨҡзҺҜеўғдёҚж”ҜжҢҒпјү


1гҖҒзј–иҜ‘е®үиЈ… python 3.10 зҺҜеўғпјҡ



2гҖҒyum е®үиЈ… ffmpeg зҺҜеўғпјҡ

  • yumе®ҳж–№дёҚж”ҜжҢҒffmpegпјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒж·»еҠ з¬¬дёүж–№жәҗ
  • ffmpeg зӣёе…іеёёз”Ёе‘Ҫд»Өпјҡ


3гҖҒйғЁзҪІе®үиЈ… faceswapпјҢ еҹәдәҺ Centos7пјҡ

  • е®ҳж–№ж–ҮжЎЈиҜҙжҳҺпјҡfaceswap/INSTALL.md at master В· deepfakes/faceswap В· GitHub
  • е®ҳж–№йЎ№зӣ®ең°еқҖпјҡGitHub - deepfakes/faceswap: Deepfakes Software For All


4гҖҒfaceswap йЎ№зӣ®иҝҗиЎҢпјҡ

  • еҗҺжңҹйғЁзҪІеҸӮиҖғпјҡжҚўи„ёејҖжәҗйЎ№зӣ®faceswapеңЁcentosзҡ„е®һзҺ°-CSDNеҚҡе®ў

  • е·ҘдҪңжөҒзЁӢ(жӯӨеӨ„иҰҒжҠҠ gakki зҡ„и„ёжҚўжҲҗ еҲҳдәҰиҸІ)пјҡ

    • 1гҖҒ/root/ai/AiSetFiles/gakkiпјҡиҰҒиў«жҚўи„ёзҡ„дәәеҺҹеӣҫпјҢеҚід»ҺзҲ¬иҷ«зҲ¬зҡ„еӣҫзүҮ
    • 2гҖҒ/root/ai/AiSetFiles/gakki_videoпјҡиҰҒиў«жҚўи„ёзҡ„дәәи§Ҷйў‘
    • 3гҖҒ/root/ai/AiSetFiles/gakki_readyпјҡиҰҒиў«жҚўи„ёзҡ„дәәпјҢеҺҹе§ӢеӣҫзүҮ/и§Ҷйў‘иҝӣиЎҢи„ёйғЁзү№еҫҒжҠҪеҸ–еҗҺпјҢж”ҫзҪ®зҡ„ж–Үд»¶еӨ№
    • 4гҖҒ/root/ai/AiSetFiles/lyfпјҡиҰҒжҚўжҲҗи°Ғзҡ„и„ёзҡ„дәәеҺҹеӣҫпјҢеҚід»ҺзҲ¬иҷ«зҲ¬зҡ„еӣҫзүҮ
    • 5гҖҒ/root/ai/AiSetFiles/lyf_videoпјҡиҰҒжҚўжҲҗи°Ғзҡ„и„ё зҡ„дәәи§Ҷйў‘
    • 6гҖҒ/root/ai/AiSetFiles/lyf_readyпјҡиҰҒжҚўжҲҗи°Ғзҡ„и„ёзҡ„дәәпјҢеҺҹе§ӢеӣҫзүҮ/и§Ҷйў‘иҝӣиЎҢи„ёйғЁзү№еҫҒжҠҪеҸ–еҗҺпјҢж”ҫзҪ®зҡ„ж–Үд»¶еӨ№
    • 7гҖҒ/root/ai/AiSetFiles/gakki_lyf_modelпјҡж”ҫзҪ®и®ӯз»ғзҡ„жЁЎеһӢж–Үд»¶
    • 8гҖҒ/root/ai/AiSetFiles/convertпјҡж”ҫзҪ®жЁЎеһӢи·‘еҮәжқҘзҡ„жңҖз»Ҳз»“жһңпјҢеҚіе·Із»ҸжҚўи„ёеҗҺзҡ„ж–Үд»¶

  • 1гҖҒжҠҪеҸ–и„ёйғЁзү№еҫҒпјҡ

  • 2гҖҒдәәе·ҘзӯӣйҖүпјҡ

    • еӣҫзүҮиҝӣиЎҢйқўйғЁзү№еҫҒжҠҪеҸ–еҗҺпјҢдјҡеҸҳжҲҗдёӢйқўиҝҷз§ҚеӨ§еӨҙз…§гҖӮ
    • иҝҷж—¶еҖҷе°ұйңҖиҰҒиҝӣиЎҢдәәе·ҘзӯӣйҖүдәҶпјҢиҝҷдёҖжӯҘжҳҜжңҖиҖ—ж—¶зҡ„пјҢеӣ дёәжңҖеҗҺе…ҘжЁЎеһӢзҡ„е°ұжҳҜиҝҷдәӣеӣҫзүҮпјҢеӣ жӯӨйңҖиҰҒд»”з»Ҷзҡ„зӯӣйҖүгҖӮ

  • 3гҖҒи®ӯз»ғжЁЎеһӢпјҡ

    • иҝҷйҮҢжңүдёӘзӮ№иҰҒйўқеӨ–жіЁж„ҸдёҖдёӢпјҢжҲ‘еңЁдҪҝз”ЁnohupиҝӣиЎҢеҗҺеҸ°иҝҗиЎҢзҡ„ж—¶еҖҷпјҢзЁӢеәҸзӣҙжҺҘеҙ©дәҶпјҢжҲ‘д№ҹжІЎжҗһжё…жҘҡжҳҜд»Җд№Ҳй—®йўҳпјҢеҰӮжһңеҮәзҺ°дәҶиҝҷз§Қжғ…еҶөпјҢе»әи®®дҪҝз”Ёjupyter-lab --port 9999 --allow-rootејҖеҗҜjupyterпјҢз„¶еҗҺеңЁжң¬ең°windowsжөҸи§ҲеҷЁжү“ејҖjupyterзҡ„з»Ҳз«ҜпјҢз„¶еҗҺеңЁиҝҷйҮҢиҝӣиЎҢжЁЎеһӢи®ӯз»ғгҖӮе°ұдёҚз”ЁжӢ…еҝғе…із”өи„‘еҗҺзЁӢеәҸиҮӘеҠЁз»ҲжӯўгҖӮ
    • жЁЎеһӢи®ӯз»ғжІЎд»Җд№ҲиҜҙзҡ„пјҢиҝҷйҮҢзҡ„жЁЎеһӢжҳҜжүӢеҠЁеҒңжӯўзҡ„пјҢиҮӘе·ұи§үеҫ—жЁЎеһӢдёҚеҶҚж”¶ж•ӣпјҢж—¶й—ҙжңүе·®дёҚеӨҡзҡ„ж—¶еҖҷпјҢзӣҙжҺҘеңЁе‘Ҫд»ӨиЎҢ ж•ІеӣһиҪҰ е°ұеҸҜд»ҘзӣҙжҺҘеҒңжӯўдәҶ

  • 4гҖҒи°ғз”ЁжЁЎеһӢиҝӣиЎҢжҚўи„ёпјҡ

    • еҜ№еӣҫзүҮиҝӣиЎҢжҚўи„ёпјҡ

    • еҜ№и§Ҷйў‘иҝӣиЎҢжҚўи„ёпјҡ

      • еҰӮжһңжғіеҜ№и§Ҷйў‘иҝӣиЎҢжҚўи„ёпјҢеҝ…йЎ»и°ғз”ЁffmpegжҠҠи§Ҷйў‘еҲҮжҲҗеӣҫзүҮпјҢз„¶еҗҺеҶҚи°ғз”ЁдёҠйқўзҡ„е‘Ҫд»ӨпјҢжңҖеҗҺеҶҚиҮӘе·ұжҠҠеӣҫзүҮжӢјжҲҗи§Ҷйў‘гҖӮж¶үеҸҠзҡ„ffmpegе‘Ҫд»ӨйғҪеңЁдёҠйқўгҖӮ
      • жҚўи„ёеҗҺз”ҹжҲҗзҡ„ж–Үд»¶жҳҜpngж јејҸпјҢеҰӮжһңffmpegж“ҚдҪңдёҚдәҶpngж јејҸеӣҫзүҮпјҢе°ұйңҖиҰҒе…ҲиҪ¬жҲҗjpg
